# Bibliotheca Explorer

Bibliotheca Explorer è un'applicazione web sviluppata con React, progettata per aiutare gli utenti a esplorare, gestire e tenere traccia dei loro libri preferiti utilizzando l'API di Google Books. Questo progetto incorpora anche Firebase per l'autenticazione degli utenti e il database in tempo reale.

## Funzionalità

- Ricerca di libri tramite l'API di Google Books.
- Visualizzazione dei dettagli dei libri, inclusi autore, descrizione e valutazioni.
- Creazione e gestione di una libreria personale per salvare i libri preferiti.
- Autenticazione degli utenti con diversi metodi (email, Google, ecc.).
- (Solo Admin) Gestione degli utenti e visualizzazione dei libri bannati.

## Tecnologie Utilizzate

- React
- Firebase (Auth, Firestore)
- Google Books API
- Axios per le richieste HTTP
- Material-UI per il design dell'interfaccia utente
